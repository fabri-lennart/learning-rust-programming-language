## ğŸš€ Quick Start

### Prerequisites
- **Rust**: Install from [rustup.rs](https://rustup.rs/)
- **Cargo**: Comes with Rust installation

### Setup
```bash
# Clone the repository
git clone https://github.com/fabri-lennart/learning-rust-programming-language.git
cd learning-rust-programming-language

# Navigate to a chapter
cd chapters/ch01-getting-started

# Create a new Rust project
cargo new hello_world
cd hello_world

# Run the program
cargo run
```
